# SaveGamePro
Save everything, then load it back.

## Features
- **Speed**
- **Built-in Types** (includes most of unity type and system types)
- **Custom Types**
- **Security**

## In Development Features
- **Web**
- **Playmaker Integration**
- **Encryption**
